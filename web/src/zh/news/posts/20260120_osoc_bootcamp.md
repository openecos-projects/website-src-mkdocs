# “一生一芯”首期国际训练营顺利结营
**第一章 又见**

与Nursultan Kabylkas教授的结缘始于2023年。彼时，他从中国科学院计算技术研究所的包云岗老师的报告中了解到“一生一芯”计划，产生了浓厚兴趣，在充分交流后，他决定在其任职的哈萨克斯坦纳扎尔巴耶夫大学开设相关课程。

2024年2月，Kabylkas教授引进了“一生一芯”计划，尝试指导学生设计处理器。但开课的过程并非一帆风顺，处理器设计对学生的动手能力要求很高，随着课程难度加深，能坚持下来的人越来越少。经过不断的试验与调整，Kabylkas教授开设了一门基础课程，引导学生打好学习基础，越来越多的学生加入到了学习芯片设计的队伍中来。他的学生Olzhas，经过一年的不懈努力，最终完成了自己的处理器芯片设计，并成功流片返回。

2025年3月，我们邀请Kabylkas教授和他的学生们来到北京，共同参与芯片调试与启动工作，见证了芯片被点亮的瞬间。这不仅是师生们的宝贵成果，更具有特殊的历史意义——这是哈萨克斯坦第一颗由本国主导设计的处理器芯片，而参与设计的学生们，也成为了该国首批处理器芯片设计人才。

![](../../res/img/web/news/20260120/02.jpg)
/// caption
2025年3月 Olzhas的芯片顺利通过测试
///

正是基于前期扎实的合作，当我们提出希望在哈萨克斯坦开办首期“一生一芯”国际训练营时，Kabylkas教授给予了全力支持。双方一拍即合，初步将时间定在2025年12月（当地寒假），时间暂定为两周。

于是，2025年12月，由“一生一芯”首席讲师余子濠博士带队，项目组三人如期抵达了纳扎尔巴耶夫大学所在地——阿斯塔纳。

**第二章 进行时**

![](../../res/img/web/news/20260120/03.jpg)
/// caption
Kabylkas教授进行开营介绍
///

本次训练营收到了90余份申请，由于时间与场地限制，Kabylkas教授最终择优录取了约40名学员。他们大多具备数字电路基础与Linux命令行操作能力，以在校大学生为主，也不乏一些业界工程师。我们后来了解到，有学员专程为课程从外地赶来阿斯塔纳，也有人为此特地申请了年假。

鉴于仅有短短两周时间，余子濠博士结合ysyx讲义，用心设计出了一套从零开始、最终完成一个具备8条指令的可流片处理器的完整课程。

![](../../res/img/web/news/20260120/04.jpg)
/// caption
余子濠老师在训练营授课
///

开营之初，学员们热情高涨，迅速投入到了高强度学习中。因为学员们基本都有些数字电路基础，所以前期的课程进行得较为顺利。

![](../../res/img/web/news/20260120/05.jpg)
/// caption
学员们讨论学习
///

然而，想要在两周内从数字电路基础学到流片，完成全流程，挑战无疑是巨大的。课程过半，进度开始出现分化。尽管大部分同学仍在坚持，但也有学员因急于求成，跳过基础任务直接挑战难点，导致调试陷入困境。我们一边答疑，一边反复强调：“慢就是快，基础打稳，比赶进度重要。”之后，大家的节奏渐渐沉稳下来，专注自己手中的代码，专注今天又学到了什么，而不是一味只顾追赶进度。

![](../../res/img/web/news/20260120/06.jpg)
/// caption
项目组工程师苏小可授课
///

为根据学员进度动态调整教学节奏，我们在保障充足自主学习时间的同时，也会持续跟进每个人的进展。既让进度领先的同学能及时获得新任务，也格外关注稍慢的同学，主动询问、及时解答。那两周里，所有人只有一个状态：在有限的时间里，不计成本，竭尽全力。

![](../../res/img/web/news/20260120/07.jpg)
/// caption
项目组工程师许立达授课
///

**第三章 未完待续**

训练营尾声，我们匿名收集了学员反馈，许多建议与我们自身总结下来的经验不谋而合。结营前的最后一刻，大家仍在争分夺秒地调试代码。但训练营的结束并非学习的终点，我们告诉学员们，项目组将会持续提供线上答疑，且流片代码不设最终提交期限。如果希望进一步深入，也欢迎学员们随时通过“一生一芯”官网（ysyx.org）的开放讲义继续学习。

![](../../res/img/web/news/20260120/08.jpg)
/// caption
训练营结营合照
///

这是一次全新的出发。我们期待未来能有更多来自世界各地的学员，从这里开始认识芯片、动手设计、并最终爱上创造芯片的旅程。我们始终相信，“一生一芯”所坚持的开源开放精神，会让这条道路通向更远、更开阔的地方。

14天的风雪无阻，我们至少一起迈出了第一步。

期待今年能继续听到来自哈萨克斯坦的好消息。
